## 6. Технические требования

### 6.1. Требования к аппаратному обеспечению

#### 6.1.1. Серверная инфраструктура
- Сервер БД: 2×Xeon Silver 4210, 64 ГБ ОЗУ, RAID 10 на SSD 1 ТБ
- Сервер приложений: 2×Xeon Silver 4210, 32 ГБ ОЗУ, SSD 500 ГБ
- Файловый сервер: 32 ГБ ОЗУ, RAID 6 на HDD 10 ТБ
- Резервный сервер: конфигурация аналогичная основному серверу приложений
- Сетевое оборудование: коммутаторы L3, маршрутизаторы, межсетевые экраны

#### 6.1.2. Оборудование сбора данных
- Шлюзы сбора данных с поддержкой multiple интерфейсов (RS-485, M-Bus, Ethernet)
- Концентраторы для беспроводных сетей (LoRaWAN, ZigBee)
- Модемы для передачи данных по GSM/LTE
- Источники бесперебойного питания для критичного оборудования

#### 6.1.3. Рабочие места пользователей
- Стандартные ПК: Core i5, 8 ГБ ОЗУ, SSD 256 ГБ
- Планшеты для мобильных сотрудников: iPad/Android с 4G
- Принтеры для печати квитанций и отчетов

### 6.2. Требования к программному обеспечению

#### 6.2.1. Серверное ПО
- ОС: Windows Server 2022 или Linux Ubuntu 20.04 LTS
- СУБД: PostgreSQL 14 с репликацией
- Веб-сервер: nginx 1.20+ или IIS 10
- Система резервного копирования: Veeam Backup & Replication

#### 6.2.2. Программные платформы
- Backend: Python 3.10+ (Django 4.1+), Java 17 (Spring Boot 3.0+)
- Frontend: React 18+, TypeScript 4.5+
- Mobile: React Native 0.70+ для кроссплатформенной разработки
- Message broker: RabbitMQ 3.11+ или Apache Kafka 3.3+

#### 6.2.3. Инструменты разработки и управления
- Система контроля версий: Git (GitLab)
- CI/CD: GitLab CI/CD или Jenkins
- Контейнеризация: Docker 20.10+, Kubernetes 1.26+
- Мониторинг: Prometheus + Grafana, Zabbix 6.0+

### 6.3. Требования к интерфейсам и интеграциям

#### 6.3.1. Внешние интерфейсы
- ГИС ЖКХ: REST API, формат обмена по утвержденным схемам
- 1С: SOAP web services, обмен в форматах XML/JSON
- Платежные системы: интеграция через API банков
- SMS/email шлюзы: REST API для массовых рассылок

#### 6.3.2. Оборудование и протоколы
- Приборы учета: MODBUS RTU/TCP, M-Bus, DLMS/COSEM, BACnet IP
- Системы телеметрии: OPC UA, SNMP для сетевого оборудования
- Системы контроля доступа: Wiegand, OSDP
